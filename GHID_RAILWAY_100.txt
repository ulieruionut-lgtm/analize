═══════════════════════════════════════════════════════════════════════════
  🆘 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

PROBLEMĂ: "npm: command not found"
SOLUȚIE:
  1. Instalează Node.js: winget install OpenJS.NodeJS.LTS
  2. SAU download: https://nodejs.org
  3. Restart PowerShell după instalare

PROBLEMĂ: "railway: command not found"
SOLUȚIE:
  1. npm install -g @railway/cli
  2. Restart PowerShell
  3. Verifică: railway --version

PROBLEMĂ: "Health check failed" după deploy
SOLUȚIE:
  1. Verifică logs: railway logs
  2. Verifică PostgreSQL e adăugat: railway open
  3. Așteaptă 2-3 minute (PostgreSQL se inițializează)
  4. Test manual: railway run python test_database.py

PROBLEMĂ: "Can't connect to database"
SOLUȚIE:
  1. Verifică PostgreSQL e creat în Railway dashboard
  2. Așteaptă 1 minut (DATABASE_URL se setează automat)
  3. Verifică: railway variables (ar trebui să vezi DATABASE_URL)
  4. Rulează din nou migrările: railway run python run_migrations.py

PROBLEMĂ: Upload PDF nu funcționează
SOLUȚIE:
  1. Verifică logs: railway logs --follow
  2. Test Tesseract: railway shell → tesseract --version
  3. Verifică PDF size (max 20 MB)
