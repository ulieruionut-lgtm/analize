┌─────────────────────────────────────────────────────────────────────────┐
│ PAS 1: Setup MySQL pe Romarg (3 min)                                   │
└─────────────────────────────────────────────────────────────────────────┘

1. Login cPanel: https://cpanel.romarg.ro
2. MySQL Databases → Create Database: "analize"
3. Create User: "analize_user" + parolă puternică
4. Add User To Database → ALL PRIVILEGES
5. Remote MySQL → Adaugă: %.railway.internal
6. Upload deployment_cpanel.zip în public_html
7. Extract + Accesează: your-domain.ro/installer.php
8. Completează date MySQL → Click "Creează Tabelele"

✓ MySQL GATA!

┌─────────────────────────────────────────────────────────────────────────┐
│ PAS 2: Deploy Python pe Railway (5 min)                                │
└─────────────────────────────────────────────────────────────────────────┘

Deschide PowerShell în: D:\Ionut analize

1. Instalare Railway CLI:
   > npm install -g @railway/cli

2. Login:
   > railway login

3. Deploy:
   > cd "D:\Ionut analize"
   > railway init --name analize-medicale
   > railway variables set DATABASE_URL="mysql://analize_user:PAROLA@cpanel.mysql.romarg.ro:3306/analize"
   > railway up

4. Obține URL:
   > railway domain

✓ APLICAȚIA LIVE!
